## Scoring Transparency

| Metric | Calculation | Range |
|--------|------------|-------|
| **Health** | Incident resolved? | 0.25 |
| **Efficiency** | steps ≤ optimal? | 0.25 |
| **Root Cause** | Identified root cause? | 0.15 |
| **Action Quality** | Wasted actions? | 0.15 |
| **Metrics** | latency < 1500ms, error < 0.1, etc. | 0.10 |
| **Sequence** | Matched correct order? | 0.10 |
| **Total** | Sum all factors | 1.0 |

**Not resolved?** Cap at 0.03 (diagnostic credit only)
